# Beginner Level Problems

This directory contains solutions for beginner-level CodeChef problems.

## Characteristics
- **Difficulty**: 1000-1400 rating
- **Focus**: Basic programming concepts, simple algorithms
- **Typical Topics**: Arrays, strings, basic math, conditionals, loops

## Solution Template

Each problem directory should contain:
- `solution.cpp` / `solution.py` / `solution.java` - Main solution file
- `README.md` - Problem description and approach
- `input.txt` - Sample input (if applicable)
- `output.txt` - Expected output (if applicable)

## Example Structure
```
ProblemCode/
├── README.md
├── solution.cpp
├── input.txt
└── output.txt
```
