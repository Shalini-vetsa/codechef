# Contest Solutions

This directory contains solutions organized by CodeChef contests.

## Contest Types

### Long Challenge
- **Duration**: 10 days
- **Problems**: 8-10 problems of varying difficulty
- **Format**: Individual contest

### Cook-Off
- **Duration**: 2.5 hours
- **Problems**: 5-6 problems
- **Format**: Individual contest

### Lunchtime
- **Duration**: 3 hours  
- **Problems**: 4-5 problems
- **Format**: Individual contest

## Organization

Each contest has its own directory with:
- Contest date or code (e.g., `COOK150`, `LTIME108`)
- Individual problem solutions
- Contest-specific README with overall performance
