# Easy Level Problems

This directory contains solutions for easy-level CodeChef problems.

## Characteristics
- **Difficulty**: 1400-1600 rating
- **Focus**: Basic data structures, simple algorithms
- **Typical Topics**: Sorting, searching, basic graph problems, number theory

## Solution Template

Each problem directory should contain:
- `solution.cpp` / `solution.py` / `solution.java` - Main solution file
- `README.md` - Problem description and approach
- `input.txt` - Sample input (if applicable)
- `output.txt` - Expected output (if applicable)
