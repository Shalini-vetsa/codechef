# Hard Level Problems

This directory contains solutions for hard-level CodeChef problems.

## Characteristics
- **Difficulty**: 1900+ rating
- **Focus**: Advanced algorithms and complex problem solving
- **Typical Topics**: Advanced DP, complex graphs, computational geometry, advanced math

## Solution Template

Each problem directory should contain:
- `solution.cpp` / `solution.py` / `solution.java` - Main solution file
- `README.md` - Problem description and approach
- `input.txt` - Sample input (if applicable)
- `output.txt` - Expected output (if applicable)
