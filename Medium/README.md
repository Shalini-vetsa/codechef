# Medium Level Problems

This directory contains solutions for medium-level CodeChef problems.

## Characteristics
- **Difficulty**: 1600-1900 rating
- **Focus**: Intermediate algorithms and data structures
- **Typical Topics**: Dynamic programming, graphs, trees, advanced math

## Solution Template

Each problem directory should contain:
- `solution.cpp` / `solution.py` / `solution.java` - Main solution file
- `README.md` - Problem description and approach
- `input.txt` - Sample input (if applicable)
- `output.txt` - Expected output (if applicable)
